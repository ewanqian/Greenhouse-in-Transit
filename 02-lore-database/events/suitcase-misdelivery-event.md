# 错投事件

- 事件名称：错投事件
- 时间：2025
- 地点：现实世界 / 展览现场
- 涉及对象：[雾庭舱](../objects/greenhouse-in-transit-case.md)
- 公开程度：内部

## 事件摘要

一件来自赛尼亚岛数字旅居地的便携温室培养舱（即雾庭舱），在空间复制中转过程中发生协议错误，未能按预定路径回到赛尼亚岛原位置，反而错误脱离数字洋流系统，掉入现实世界。

## 已知经过

1. 原旅客将植物性记忆样本封存进便携温室舱
2. 转栈署接收容器，安排复制运输回赛尼亚岛
3. 复制协议发生错误，容器坐标偏移
4. 容器错误脱离系统，出现在现实世界展览现场
5. 转栈署记录异常，等待回收

## 未确认部分

- 原旅客是否仍在赛尼亚岛停留
- 容器内封存的样本是否完整
- 是否还有其他容器发生类似错误
- 错误投递是否有意为之
