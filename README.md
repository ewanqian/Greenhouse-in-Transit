# Greenhouse in Transit / 箱中温室

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-blue)](https://ewanqian.github.io/Greenhouse-in-Transit/)

> **这不是一只普通箱子，而是一座被错投到现实的温室。**

《箱中温室》是一件以手提箱为现实入口的空间叙事项目。它表面像一件来自数字旅居地的遗失行李，深层上却是一只被命名为"雾庭舱"的便携温室舱：原本属于赛尼亚岛中的一位长期驻留旅客，在一次中转错投后掉入现实。

## 项目概述

这件作品从一只银色手提箱开始，但目标不是展示"箱子"本身，而是让观众相信：有一种空间确实可以被携带、被中转、被错误投递，然后在现实中被重新开启。

观众先看到实体箱体，再从手机端进入项目主页，理解赛尼亚岛、松间、白屿会与雾庭舱之间的关系；之后通过档案查看器进一步读到错投记录、旅客线索和物件说明，最后在 Vision Pro 中短暂看到一座被压缩的温室从箱体内部展开。

## 命名体系

项目采用双层命名结构：

- **对外**：箱中温室 / Greenhouse in Transit —— 项目题名，面向公众与策展语境，保留诗性。
- **对内**：雾庭舱 / Mistgarden Case —— 物件名，项目内部真正的装备级识别名，型号 MG-17 Portable Conservatory Unit。

外层旅居品牌为松间，由白屿会持有，通过潮线网络航行，异常回收由转栈署负责。

## 世界观架构

### 表层：松间 / SONGJIAN
松间是赛尼亚岛的外层旅居品牌。它对外呈现地图、空间节点、景观、停留方式与品牌物料，让公众把这座岛理解为一处高级、静谧、可长期停靠的数字旅居地。

### 深层：数字洋流与潮线
赛尼亚岛同时属于更大的数字航行宇宙。旅客、维护员与各类舱体通过数字洋流迁移，而"潮线"负责航行、接驳与路线网络。空间不只是目的地，也是被运输的对象。

### 异常：雾庭舱
这只箱体原本是赛尼亚岛中的一件便携温室舱，由一位长期驻留旅客持有。因为一次错投，它脱离原路径进入现实，成为展览现场能够被观者近身接触的一件异常证物。

## 体验流程

1. **远看**：观众先看到一只银色手提箱。它在"登机仪式"语境里成立，同时又因为标签、编号与异常质感而显得不像普通展具。
2. **近看**：箱内外可见票据、参数卡、洗漱套件、样本袋、品牌说明、遗失提示等小型证物，强化"旅居 / 驻留 / 错投"三重语义。
3. **扫码**：手机进入项目主页，先理解赛尼亚岛、松间与雾庭舱；再进入档案查看器，逐步发现转栈署记录和旅客线索。
4. **观看**：Vision Pro 中，雾庭舱会在 60–90 秒内展开成一座临时温室：球壳、线框、点云植物、虹彩圆盘与花园残影从箱体内显影。
5. **留存**：观众带走票据、小物或气味卡，离场后再通过网页继续追踪项目，形成 ARG 式的延迟体验。

## 关于 VIRTURA

VIRTURA 数字航行宇宙是一个由数字空间、目的地系统、记忆封存、叙事物件和现实侵入机制共同构成的开放宇宙。在这个宇宙中，空间不只是环境，也可以被携带；景观不只是背景，也可以被封存在容器中。

**核心理念**：空间既是目的地，也是被运输的对象。
