# 项目主页大纲

- 给策展人介绍项目

- 给 GitHub 项目做 landing page
- 白底、清晰、专业、美术馆级项目页

## 信息顺序

```
1. 标题区
2. 项目一句话
3. 项目主图 / 短视频
4. 项目说明
5. 为什么是手提箱
6. 为什么连接赛尼亚岛数字旅居地

8. 视觉语言
9. 当前版本状态
10. 相关实验路径（只占一到两成）
11. 后续发展
12. 联系 / 仓库入口
```
