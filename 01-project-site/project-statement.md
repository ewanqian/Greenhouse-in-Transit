# 项目说明

## 核心概念

《箱中温室》的核心不在于"展示一个箱子"，而在于"展示一个箱子如何展开成空间"。

箱体不是普通容器，而是一种便携式空间舱——来自数字洋流潜航系统中的旅客随身配置，因一次复制协议错误，被错误投递到现实世界。

因此，这件作品同时具有三种身份：

1. **它是一件实体遗失物** —— 观众在展览现场遇到一只经过改造的手提箱，带有运输痕迹、标签和未来设备感，本身即可作为独立展览对象
2. **它是一段被压缩封存的空间记忆** —— 内部储存着植物性记忆、环境样本、局部温室结构和一段被压缩的驻留空间
3. **它是一个跨介质入口** —— 连接现实、网页、Minecraft、AR 与 Vision Pro 体验，是赛尼亚岛宇宙错误侵入现实的接缝点

## 为什么是手提箱

手提箱天然携带"旅行、运输、遗失、携带"这些叙事元素。它符合项目设定中"便携空间容器"的物理形态，也符合展览"艺术家携带作品到场"开幕仪式的要求。

箱子本身就是一种邀请：观众会自然好奇"箱子里有什么"，这种好奇心正好驱动他们完成从扫码到深入阅读的完整旅程。

## 为什么连接赛尼亚岛数字旅居地

《箱中温室》属于"赛尼亚岛数字旅居地"宇宙中的一个异常入口项目。

赛尼亚岛数字旅居地表面上是一个模拟挪威峡湾景观的虚拟度假区，对外以北欧自然旅居目的地的姿态出现，并通过"松涧目的地"这一旅居品牌进行运营。用户可以通过地图、导览文案、品牌物料、Minecraft 入口和场景介绍来理解这座岛屿的空间分区与旅居方式。

但在更深层的叙事中，赛尼亚岛同时也是数字洋流潜航系统中的一个目的地节点，一些旅客、研究员、长期驻留者和空间维护者会携带各类便携空间容器、记忆封存箱和环境培养仓进入其中。《箱中温室》就是其中一件因复制错误而脱离原系统、掉入现实的便携温室舱。

这使得《箱中温室》成为赛尼亚岛项目中最适合作为 ARG 切口和空间计算展示切口的部分：它既能为赛尼亚岛增加一个富有戏剧性的"异常入口"，也能为整个数字旅居地项目增加一层更深的宇宙逻辑。

## 体验结构

| 层面 | 体验内容 |
|------|----------|
| **实体层面** | 观众看到一只经过改造的遗失手提箱，带有运输痕迹与标签 |
| **手机层面** | 扫码进入失物登记页面，分流到公开说明 / 认领流程 / 档案入口 |
| **Vision Pro 层面** | 观者看到箱体内部逐渐展开出完整的温室空间动画 |
| **档案层面** | 可在线浏览完整世界观数据库，深入阅读所有词条 |
