# 视觉语言说明

## 设计原则

- **白底**：项目主页用白底，清晰专业，符合美术馆展示要求
- **sharp**：无衬线字体，干净的边框，信息层次分明，不炫技
- **专业**：像产品介绍页，直接回答问题，不绕弯子
- **不割裂**：世界观数据库用黑底，像现代开发者文档/数据库页面，蓝色强调，整体统一

## 配色方案

### 项目主页（对外）
- 背景：白色 `#ffffff`
- 主文本：深灰 `#111111`
- 次级文本：中灰 `#666666`
- 边框：浅灰 `#e0e0e0`
- 强调色：蓝色 `#2563eb`

### 档案查看器（对内）
- 背景：深黑 `#0a0a0a` / `#0f1117`
- 主文本：白色 `#f0f6fc`
- 次级文本：中灰 `#8b949e`
- 边框：深灰 `#30363d`
- 强调色：蓝色 `#58a6ff`

## 为什么是这种视觉

这个项目不是要做一个传统艺术展览海报，也不是要做一个花里胡哨的赛博朋克页面。

它要做的是：
- 对外：清晰专业地说明项目，让策展人和观众快速理解
- 对内：像一个真实存在的数据库/档案系统，增强 ARG 浸入感

所以我们选择：
- 项目主页 → 白底 + 无衬线 + 清晰卡片 → 美术馆级展示
- 档案查看器 → 黑底 + 现代开发者风格 → 模拟真实数据库，增强"这里真的有一个宇宙系统"的感觉

## 元素层级

视觉识别度来自多种层次的叠加，不是单一风格：

| 层级 | 元素 | 作用 |
|------|------|------|
| 基底 | 北欧自然地貌 | 提供静谧旅居氛围 |
| 氛围 | 东方静谧空间感 | 提供呼吸感与留白 |
| 结构 | 线框与几何壳体 | 提供系统感与切割 |
| 图层 | 漂浮代码与系统 UI | 提示数字背景 |
| 气质 | 复古数字语言 / Y2K | 提供锋利度 |
| 质感 | 虹彩圆盘 + 半透明板 | 提供光感 |
| 内容 | 高斯植物 + 点云花园 | 提供有机内容 |
