# 策展介绍文案

## 一句话

一只来自数字航行系统的遗失手提箱，内部封存着一段植物性记忆温室；它因空间复制错误掉入现实，并通过实体物件、网页档案、手机端线索与头显观看逐渐展开。

## 策展说明

《箱中温室》是 VIRTURA 数字航行宇宙中的一件异常物，它测试"单一物件如何成为空间入口"这一命题。

在数字航行宇宙的设定中，空间可以被压缩封存，装入便携容器，通过数字洋流运输。这件作品就是这样一件容器——它原本属于赛尼亚岛数字旅居地的一位长期驻留旅客，因一次复制错误，错误脱离系统，掉入现实世界。

观众在展览现场遇到这只箱子，可以：

- 近看物件细节、标签与材质
- 扫描二维码进入手机端档案入口
- 通过 Apple Vision Pro 观看箱体内部空间逐渐展开的动画
- 在线浏览完整世界观数据库

作品不追求重型交互系统，而是通过"实体物件 + 分层叙事 + 头显动画"的组合，让观众亲身经历"一个从遗失物到展开空间"的过程。它既是一件展览作品，也是一个持续扩展的 ARG 入口，更是 VIRTURA 宇宙方法论的一次直观演示：**空间既是目的地，也是被运输的对象。**
