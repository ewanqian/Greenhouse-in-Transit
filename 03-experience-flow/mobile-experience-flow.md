# 手机端体验流程

## 用户扫码后的四层结构

### 页面 1：识别页

你拾获的是：
**一件异常登记中的便携空间容器**

按钮：
- 查看公开说明
- 进入认领流程
- 浏览旅居地资料

### 页面 2：公开说明

偏项目主页简版，包含：
- 项目一句话介绍
- 照片
- 入口链接到完整项目主页
- 入口链接到档案查看器

### 页面 3：认领流程

不是现实认领，而是：
- 请输入箱体编号（箱体表面印有）
- 你在哪看到它
- 你是否接触过内部物件
- 你是否闻到特殊气味

收集这些信息用于 ARG 后续扩展。

### 页面 4：解锁页

根据选择进入不同档案页，完成解锁。
