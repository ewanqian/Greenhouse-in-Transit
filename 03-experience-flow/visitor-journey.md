# 观众完整体验旅程

## 体验总流程

### Stage 0：远看
用户先看到一只箱子，知道它是本次唯一媒介之一。这和展览整体规则匹配。

### Stage 1：近看
用户发现箱子不是单纯展示物，有标签、票据、物件、说明、气味、遗失感。

### Stage 2：读取
用户看到某个二维码 / 编号 / 失物说明。

### Stage 3：扫码
进入手机网页，不是直接跳长文，而是进入"失物识别页 / 档案入口页"。

### Stage 4：分流
用户可以选：
- **公开项目介绍** → 直接看项目说明
- **失物认领入口** → 进入认领流程
- **旅居地地图** → 浏览赛尼亚岛信息
- **档案阅读器** → 深入浏览所有世界观词条
- **乘客记录** → 如果认领，查看关联记录

### Stage 5：深入
用户开始从表层"旅居品牌"进入深层"异常遗失"。

### Stage 6：扩展
部分用户继续进入：
- **Vision Pro 短体验** → 观看空间展开动画
- **假官网 / 假系统入口** → 浏览更多机构页面
- **票据与物件扫描页** → 扫描箱体上的各个标签
- **后续 ARG 更新页** → 订阅后续更新

## 用户分层

### 用户 A：策展人 / 合作方
- 目标：快速理解项目
- 入口：[项目主页](../project.html)
- 路径：项目主页 → 相关实验 → 联系

### 用户 B：普通现场观众
- 目标：进入体验
- 入口：箱体 → 扫码 → 手机
- 路径：扫码 → 识别页 → 公开说明 / 认领流程 → 档案

### 用户 C：重度世界观用户
- 目标：深挖 lore
- 入口：[档案查看器](../archive.html)
- 路径：分类浏览 → 词条阅读 → 关联跳转

## 现场策略

**现场不要像一整个宇宙说明会**，而是：
> **一只箱子 + 少量关键物 + 明确扫码入口 + 极少量文本**

真正厚重的世界观，都进入手机和档案查看器。
